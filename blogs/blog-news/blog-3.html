Jquery automatically gives header height when using position fixed

===
Code Jquery 
$("header").height($(".header_class").outerHeight() + 1);
Trong đó header là div cần dùng để tự canh chiều cao Class ".header_class" chứa header fixed

See the Pen js auto height header fixed by Tào Lao (@gifuitvnluan) on CodePen.
